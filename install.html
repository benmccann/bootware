# Installation

Bootware is invoked by shell scripts on the user's computer. The following commands will download the shell scripts and add them to the system path.

On an Apple Silicon system, ensure that the current terminal is not running under Rosetta 2, by confirming that the output of command uname -p is arm.

On Windows, PowerShell will need to run as administrator and the security policy must allow for running remote PowerShell scripts. The following command will update the security policy, if needed.

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine
