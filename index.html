# Bootware

Documentation: https://wolfgangwazzlestrauss.github.io/bootware

Source Code: https://github.com/wolfgangwazzlestrauss/bootware

Bootware is a set of shell scripts and Docker images for bootstrapping software installations with Ansible.

# Install

For instructions on installing Bootware for your operating system, see the Installation (opens new window) section of the documentation.

# Usage

Bootware will bootstrap the computer software by invoking bootware bootstrap after customizing the configuration file. To view the bootstrapping options, execute bootware bootstrap --help.
